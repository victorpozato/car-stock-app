# CarStockApp

Aplicativo mobile para gestão de estoque de veículos em lojas.

## Funcionalidades

- Login de usuários
- Visualização de veículos em formato de card
- Tela detalhada com informações completas do veículo
- Sistema de permissões por Usuarios....

## Tecnologias

- React Native
- SQLite
